How does ﻿InvestingPro+ ﻿calculate fair value?

Investing Support
  • July 10, 2026 01:33
  • Updated
Follow

Understanding the fair value of a potential investment is crucial for making informed financial decisions. InvestingPro+ offers a sophisticated approach to fair value calculation that provides users with comprehensive insights into asset valuation. This article explores how InvestingPro+ determines fair value and how you can leverage this feature in your investment research process.

The Fair Value Calculation Methodology

InvestingPro+ calculates fair value using more than 15 proven financial models, providing a multi-faceted approach to valuation. This comprehensive methodology allows for a more robust assessment compared to relying on a single valuation technique. By incorporating various models, InvestingPro+ delivers a more nuanced perspective on an asset's potential worth.

Financial Models Used in Fair Value Calculations

Investing.com employs a diverse range of financial models to determine fair value, including:

Multiple-Based Models

  • EBITDA Multiples: Compares enterprise value to earnings before interest, taxes, depreciation, and amortization
  • Revenue Multiples: Evaluates company value relative to its revenue
  • EBIT Multiples: Assesses company value compared to earnings before interest and taxes
  • P/S Multiples: Price-to-sales ratio comparison
  • P/E Multiples: Price-to-earnings ratio analysis
  • P/B Multiples: Price-to-book value assessment

Dividend-Based Models

  • Dividends: Stable Growth: Valuation based on consistent dividend growth patterns
  • Dividends: Multi-Stage: Analysis incorporating varying dividend growth rates over different time periods

Discounted Cash Flow (DCF) Models

  • 5Y DCF Revenue Exit: 5-year discounted cash flow with revenue-based terminal value
  • 5Y DCF EBITDA Exit: 5-year DCF with EBITDA-based exit value
  • 5Y DCF Growth Exit: 5-year DCF with growth-rate terminal value
  • 10Y DCF Revenue Exit: 10-year projection with revenue-based terminal value
  • 10Y DCF EBITDA Exit: 10-year projection with EBITDA-based exit
  • 10Y DCF Growth Exit: 10-year projection with growth-rate terminal value

Additional Models

  • Earnings Power Value: Focuses on sustainable earnings capacity

Customizable Simulation Capability

One of the most powerful aspects of InvestingPro+'s fair value calculator is its customization feature. Users can:

  • Simulate changes to fair value by adjusting potential valuations
  • Test different scenarios to understand how various factors might impact an asset's worth
  • Use these simulations to help visualize possible future outcomes based on different assumptions

This simulation capability provides users with a dynamic tool for exploring various market conditions and company performance scenarios, enhancing their understanding of potential valuation changes.

 

Utilizing Fair Value Information Effectively

The fair value calculations provided by InvestingPro+ serve as informative data points that can contribute to your broader research process. By accessing these calculations, you gain additional context that can help you better understand asset valuations from multiple perspectives.

It's worth noting that these fair value calculations are based on mathematical models and historical data analysis, offering valuable insight into possible valuations based on different methodologies. These calculations represent one of many tools available to InvestingPro+ users for conducting thorough financial analysis. 

Note: Fair value calculations are estimates based on models and historical data, and are not guarantees of future performance or definitive buy/sell signals.
